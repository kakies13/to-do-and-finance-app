## ğŸ“– Usage

### Getting Started
1. Launch the application
2. Set your initial balance in the Dashboard
3. Start adding notes and transactions
4. Create categories to organize your finances
5. Set up installments for recurring payments

### Key Features Guide

#### Notes
- Click the "Notes" section in the sidebar
- Click "Add Note" to create a new note
- Set alarms by clicking the alarm icon on any note
- Edit or delete notes using the action buttons

#### Transactions
- Navigate to "Transactions" section
- Add income or expense transactions
- Select categories and add descriptions
- View transaction history with filters

#### Installments
- Go to "Installments" section
- Add new installment plans
- Track payment status month by month
- Mark installments as paid

#### Dashboard
- View your current balance
- See recent transactions
- Check upcoming notes with alarms
- Monitor financial statistics
